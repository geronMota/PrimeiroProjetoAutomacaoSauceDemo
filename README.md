
# 🚀 Projeto Final – Testes Automatizados com Selenium

Este projeto foi desenvolvido como atividade final do curso de automação de testes, com o objetivo de simular um cenário real de QA Automation, utilizando boas práticas de organização, legibilidade e manutenção de código.

---

- Jefferson Mota Silva  
- Elton Ferreira Lana

---

## 🎯 Objetivo

Desenvolver uma suíte de testes automatizados utilizando **Selenium WebDriver**, simulando um ambiente profissional de testes, com foco em:

- Organização do código  
- Reutilização de componentes  
- Clareza e boas práticas  
- Estrutura escalável  

---

## 🧪 Sistema Testado

Foi utilizado um sistema web de demonstração com funcionalidades comuns de aplicações reais, como:

- Login de usuário  
- Navegação entre páginas  
- Interações com formulários  
- Validação de mensagens  

---

## ✅ Cenários Automatizados

Foram implementados múltiplos cenários de teste, incluindo:

- ✔️ Login válido (teste positivo)  
- ❌ Login inválido (teste negativo)  
- 📝 Interação com formulários  
- 🔍 Navegação entre páginas  
- ⚠️ Validação de mensagens de erro
